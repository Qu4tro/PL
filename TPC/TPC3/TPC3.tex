\section{Expressões Regulares e Autómatos}

Considere as seguintes ERs:

    $e1 = a + (c^{*} + b^{*})$
    
    $e2 = a a^{*} b $
    
    $e3 = a + a^{*} b $
    
    $e4 = a + (a b)^{*} $
    
    $e5 = c +\ (a + b)^{*} $

e construa formalmente o Autómato Não-Determinista equivalente a cada ER.
